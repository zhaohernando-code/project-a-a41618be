# 一个关于a股的当前数据和投资建议看板

希望有如下功能
1.  选中股票的行情图看板
2. 对选中股票建立用历史数据金融模型（这方面我不了解，但希望你能找到业界的比较新的、成熟的方案）。用来趋势预测
3. 结合板块新闻，提取出因子纳入考量
4. 结合2，3，支持点击后拉起gpt给出当下投资建议（购入、卖出、观望等）
5. 优质股票推荐
6. 模拟交易
7. 整体输出业外人士友好，能看懂
8. 可以用数据库存一些数据
9. 优先参考github或者别的平台有没有符合一部分需求的开源项目，可以囊括一部分然后迭代开发
以上为一个行业外的视角，我希望你在我写的基础上，理解我的诉求（核心是投资建议），附上一些我想不到但很重要的功能点
涉及到的具体的计算、建模等方案由你来定，我不懂，但希望够权威和专业
